# TASK-51 — Initialize project with Vite + React + TypeScript

## Overview

Initial project setup using Vite with React and TypeScript.

## Steps

```bash
npm create vite@latest my-app -- --template react-ts
cd my-app
npm install
npm run dev
```

## Tech Stack

- **Vite** — Build tool
- **React** — UI library
- **TypeScript** — Type safety

## Outcome

Project bootstrapped and running on `http://localhost:5173`
